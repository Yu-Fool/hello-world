## Git提交记录

+ `git init`
  
>初始化

+ `git add .`

>添加所有的文件（后面是一个英文的句号）
>也可以不用句号，用具体的文件名

+ `git commit "这里写下你自己的记录本次提交内容的信息"`

>执行提交(后面加双引号内是备注)

+ `git push git@github.com:自己的名称/仓库名.git`

我自己的自然就是
git push git@github.com:Yu-Fool/hello-world.git

>执行推送

***每次增加了新文件就先 add，然后 commit，如果只是改了文件的内容，只执行commit就行了，最后一步都是要执行 push 。***
